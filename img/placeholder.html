Portfolio Image Placeholders

This file contains instructions for adding portfolio images to your website.

Portfolio Images

For the portfolio section, you'll need to add 6 images named:

  • portfolio-1.jpg
  • portfolio-2.jpg
  • portfolio-3.jpg
  • portfolio-4.jpg
  • portfolio-5.jpg
  • portfolio-6.jpg

These should be placed in the img/ directory.

Image Recommendations

For best results:

  • Use high-quality images that represent your work
  • Recommended size: 800x600 pixels
  • Keep file sizes under 300KB for optimal loading
  • Use JPG format for photographs
  • Use PNG format for graphics with transparency

Placeholder Resources

Until you have your own images, you can use placeholder services like:

  • placeholder.com
